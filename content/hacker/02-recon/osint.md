开源情报收集利用公开可获取的信息来了解目标，是最隐蔽的信息收集方式。

## 搜索引擎情报

### Google Hacking
- 高级搜索语法（site、inurl、intext、filetype）
- 敏感信息泄露（配置文件、日志、备份）
- 目录列表和文件暴露
- 错误信息和调试页面

### 专业搜索引擎
- Shodan（物联网设备、工控系统）
- Censys（证书、服务、配置）
- Fofa（网络空间资产）
- ZoomEye（网络空间测绘）

## 代码仓库情报

### GitHub 搜索
- 泄露的密钥和凭证
- 配置文件和敏感信息
- 内部项目和代码
- 员工账号和邮箱

### 其他平台
- GitLab、Bitbucket
- Pastebin、Gist
- 代码分享网站
- 开发者社区

## 社交媒体情报

### 人员信息
- LinkedIn（员工、职位、技能）
- Twitter、微博（动态、关系）
- Facebook、Instagram
- 专业论坛和社区

### 组织信息
- 公司架构和部门
- 技术栈和项目
- 合作伙伴和客户
- 活动和会议

## 企业情报

### 工商信息
- 企业注册信息
- 股东和高管
- 分支机构
- 关联企业

### 招聘信息
- 技术栈和工具
- 系统架构
- 安全产品
- 人员需求

### 公开文档
- 财报和公告
- 技术白皮书
- 产品文档
- 新闻报道

## 域名与网络情报

### WHOIS 信息
- 域名注册信息
- 联系人信息
- DNS 服务器
- 历史记录

### DNS 记录
- A、AAAA、MX、TXT 记录
- SPF、DKIM、DMARC
- 子域名信息
- IP 地址段

### 网络资产
- IP 归属和地理位置
- AS 号和网络范围
- 路由信息
- 网络拓扑

## 暗网情报

### 暗网论坛
- 黑客论坛和社区
- 漏洞交易市场
- 工具和服务
- 威胁情报

### 数据泄露
- 凭证数据库
- 邮箱和密码
- 数据库转储
- 内部文档

## 情报分析

### 信息关联
- 人员关系网络
- 资产关联分析
- 时间线构建
- 模式识别

### 威胁建模
- 攻击面识别
- 风险评估
- 攻击路径规划
- 优先级排序

---

OSINT 收集完全基于公开信息，不会触发目标的任何防护，是最安全的信息收集方式。
