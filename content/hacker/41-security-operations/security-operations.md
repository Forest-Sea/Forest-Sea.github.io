SOC 运营、SIEM、日志分析、安全监控、事件响应流程。

## SOC 运营

### SOC 架构
- L1 分析师（监控和分类）
- L2 分析师（深入分析）
- L3 分析师（威胁狩猎）
- 事件响应团队
- 威胁情报团队

### 运营流程
- 监控和检测
- 事件分类
- 事件分析
- 事件响应
- 事件关闭
- 总结和改进

- AlienVault

- 实时监控

- 告警生成
- 事件调查
- 报告和仪表板

### 规则和关联
- 规则编写
- 关联规则
- 威胁检测
- 异常检测
- 基线建立
