C2（Command and Control）木马和后门的免杀技术，用于绕过杀软、EDR 和安全检测。

## 免杀核心

### 静态特征规避
- 去除明显的 API 调用特征
- 字符串加密（所有敏感字符串）
- 代码混淆（控制流平坦化、虚拟机保护）
- 资源节伪装（图标、版本信息、数字签名）
- 熵值控制（避免高熵特征）

### 动态行为规避
- API 动态解析（GetProcAddress、LoadLibrary）
- 系统调用（直接 syscall、间接 syscall）
- API Hook 绕过（Unhook、直接系统调用）
- 内存操作隐藏（VirtualAlloc、VirtualProtect）
- 进程行为伪装

### 行为检测对抗
- 沙箱检测（虚拟机、调试器、分析工具）
- 延迟执行（Sleep、时间检测）
- 环境检测（用户交互、系统信息）
- 父进程检查（PPID Spoofing）

## Shellcode 处理

### 编码技术
- XOR 编码
- AES/RC4 加密
- 自定义加密算法
- 多层加密嵌套
- 分段加密

### 加载技术
- 直接内存执行
- 进程注入（远程线程、APC、进程镂空）
- 模块踩踏（Module Stomping）
- 反射 DLL 注入
- 无文件加载

### 分离免杀
- 加载器和 Payload 分离
- 加载器负责解密和执行
- Payload 完全加密存储
- 远程加载（HTTP、DNS）

### 生成策略
根据目标环境的防护强度选择：
- 轻度防护：基础混淆 + 简单加密
- 中度防护：代码混淆 + 多层加密 + 行为伪装
- 重度防护：完全加密 + 虚拟机保护 + EDR 对抗 + 无文件执行
