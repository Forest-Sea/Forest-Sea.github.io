## 认识成瘾循环（《跳出色情陷阱》核心理论）

破戒不是偶然事件，而是按照一个固定循环进行的：

```
沉迷（想入非非）→ 仪式化（准备动作）→ 行为（破戒）→ 绝望（内疚发誓）→ 再沉迷
```

**打破循环的关键点：**
- 🌟 **沉迷期打断**：刚有念头时，立刻转移注意力，做别的事
- 🌟 **仪式化期打断**：当你发现自己正在做"准备工作"时（锁门、拉窗帘、打开特定APP），立刻停手离开现场
- 🌟 **不要在绝望期发誓**：刚破戒的内疚感会让你做出不切实际的承诺，冷静下来再制定计划

## 《停止性瘾》的"断绝"实战法

一旦意识到欲望正在上升，立即执行：

**3 秒法则**：
1. 第 1 秒——身体离开当前位置（站起来、走出房间）
2. 第 2 秒——做剧烈动作（用冷水洗脸、做俯卧撑）
3. 第 3 秒——联系问责伙伴或打开戒色文章

## 破戒的常见类型

### 类型一：无聊破戒
- 症状：没事做的时候会想起色情
- 应对：提前安排好时间，不要让大脑处于"放空"状态
- 预防：每天列计划表，把时间填满

### 类型二：压力破戒
- 症状：学习/工作压力大时，用色情来逃避
- 应对：找到健康的减压方式（运动、听音乐、散步）
- 预防：建立正面应对压力的能力

### 类型三：独处破戒
- 症状：一个人独处时，心魔容易趁虚而入
- 应对：保持感恩心和警觉心，独处时更要加强观心
- 预防：睡觉不锁门、不在床上玩手机

### 类型四：睡前破戒
- 症状：睡前躺着刷手机，不小心看到擦边内容
- 应对：睡前半小时放下手机，放远一点充电
- 预防：卧室不放手机，用闹钟代替

### 类型五：情绪破戒
- 症状：心情不好、生气、失落时找安慰
- 应对：认识情绪，处理好自己的负面情绪
- 预防：培养积极心态，学会情绪管理

### 类型六：擦边破戒
- 症状：看了擦边内容，然后一步步陷进去
- 应对：擦边就是破戒的前兆，必须远离
- 预防：主动过滤手机内容，取消关注不良博主

## 断念实战（针对不同类型的念头）

### 1. 图像型念头
脑子里出现色情画面
→ 立刻转移注意力，做几个俯卧撑或站起来走几步
→ 默念断念口诀

### 2. 回忆型念头
想起以前看过的色情内容
→ 告诉自己"那是过去了，我不需要回望"

### 3. 诱惑型念头
"就看一眼，没事的"
→ 这是心魔在撒谎！每次破戒前都是这么想的

### 4. 合理型念头
"今天压力大，放松一次"
→ 放松的方式有很多，不要选最差的那种

## 心魔的套路

心魔就像一个狡猾的对手，它常用的招式：

1. **伪装成"最后一次"** — "我这次确实没控制住，下次一定断"
   → 没有下一次，这次就得断

2. **合理化** — "适度很正常，所有人都会"
   → 前面说了，适度无害论是100%错误的

3. **利用疲劳** — "你太累了，看个视频放松一下"
   → 睡觉才是最好的放松

4. **利用情绪** — "心情不好，就看这一次"
   → 心情不好更要保持自制

5. **利用空虚** — "你什么也没干，不如..."
   → 无聊的时候就去学习、运动

6. **计时器陷阱** — "坚持X天了，奖励自己一次"
   → 坚持本身就是最大的奖励，不需要"奖励"自己破戒

## 实战中的黄金时间

当淫念来袭时，**前5秒是黄金时间**：
- 第1秒 — 觉察到念头来临
- 第2秒 — 立刻念断念口诀
- 第3秒 — 做物理动作（起身/运动/洗脸）
- 第4秒 — 转移注意力到别的事情
- 第5秒 — 如果做到了，高峰过去，念头减弱

**超过5秒**还在犹豫，破戒概率大大增加。

## 深夜奋起法

如果在深夜突然涌起欲望：
1. 立刻**从床上坐起来**，不要躺着抵抗
2. 下床走动，去喝水/洗脸
3. 做固肾功30个
4. 看5分钟戒色文章
5. 欲望高峰期（10-30分钟）过去就好了

## 破戒后的处理

### 不要做的三件事
1. **不要自暴自弃** — "反正已经破了，明天再说"
2. **不要破罐子破摔** — 连续破戒
3. **不要过度沉浸在内疚中** — 内疚但没有行动改变

### 要做的事
1. **总结经验** — 为什么会破？什么情境？什么时间？
2. **调整策略** — 针对性地调整自己的戒色方案
3. **重新出发** — 总结经验后放下包袱继续戒

## 戒色禁忌

1. **不意淫** — 意淫也是邪淫，会消耗精气神
2. **不看擦边** — 不看任何擦边内容（短视频、社交媒体）
3. **不久坐** — 久坐伤肾，且容易引起欲望
4. **不吃刺激性食物** — 辛辣、油腻容易扰动欲望
5. **不睡懒觉** — 早起避淫，阳气充足
6. **不裸睡** — 减少物理刺激
7. **不上不良网站** — 直接屏蔽，不给自己找借口
