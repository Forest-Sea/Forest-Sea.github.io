## 写文章的步骤

一共三步，每次都是这样：

### 第一步：新建文件

在 VS Code 里，右键 `content/posts/` 文件夹 → 新建文件 → 取个英文名，比如 `my-post.md`

### 第二步：写内容

文件开头必须有这段元数据（这是文章的基本信息）：

```markdown
---
title: "文章标题"
date: 2026-05-27
tags: ["标签1", "标签2"]
summary: "文章摘要，会显示在首页"
---

这里开始写正文。
```

### 第三步：推送

打开终端，输入三行命令：

```bash
git add .
git commit -m "新文章：文章标题"
git push
```

等 30 秒，刷新网站，文章就出现了。

## Markdown 常用语法

| 写法 | 效果 |
|------|------|
| `## 标题` | 二级标题 |
| `**加粗**` | **加粗** |
| `- 列表项` | 无序列表 |
| `[链接文字](网址)` | 超链接 |
| `` `代码` `` | 行内代码 |
