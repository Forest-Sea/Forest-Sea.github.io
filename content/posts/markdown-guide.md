## 标题

用 `##` 开头的就是二级标题，`###` 是三级标题

## 正文

正文直接写字就行。空一行就是新段落。

多个连续空行会被压缩成一个段落间距，不用担心。

## 加粗和斜体

**这是加粗的文字**，*这是斜体的文字*。

## 列表

无序列表用 `-` 开头：

- 第一项
- 第二项
- 第三项

有序列表用数字：

1. 第一步
2. 第二步
3. 第三步

## 链接

格式是 `[显示的文字](网址)`，例如：

[百度](https://www.baidu.com)

## 代码

行内代码用反引号包起来：`console.log("hello")`

代码块用三个反引号包起来：

```python
print("Hello World")
for i in range(5):
    print(i)
```

```bash
git add .
git commit -m "消息"
git push
```

> 思而不学则殆

## 分隔线

上面一条横线就是分隔线，用三个短横 `---`
